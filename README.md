Алгоритм — это точная последовательность действий для решения задачи.

Свойства алгоритма: конечность, определённость, результативность, массовость.

Линейный алгоритм выполняется строго по порядку, с ветвлением — выбирает путь по условию.

Циклы нужны для многократного повторения действий.

Типы циклов: с предусловием, с постусловием, с параметром.
